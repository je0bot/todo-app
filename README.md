# Todo List CLI App 📝

A simple command-line Todo List application built using Python.  
It allows users to add, view, and manage tasks.

---

## 📌 Description

This is a beginner-friendly CLI todo app built to practice Python fundamentals and Git workflow.

---

## 🚀 Features

- Add new tasks  
- View all tasks  
- Mark tasks as completed  
- Delete tasks  

---

## 🛠️ Tech Used

- Python  

---

## ▶️ How to Run

```bash
python todo.py
