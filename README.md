# Todo List CLI App 📝

A simple command-line Todo List application built using Python.
It allows users to add, view, and manage tasks.

---

## 🚀 Features

* Add new tasks
* View all tasks
* Mark tasks as completed
* Delete tasks

---

## 🛠️ Tech Used

* Python

---

## ▶️ How to Run

```bash
python todo.py
```

---

## 📂 Project Structure

* `todo.py` → main application
